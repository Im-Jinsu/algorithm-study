## 계산 메서드
#### Math.sqrt(number)
number의 제곱근을 구해 반환한다. 음수값은 매개변수로 사용하지 못한다.
매개변수 타입: int, double, long <br> 
#### Math.pow(밑, 지수)
밑의 지수 제곱근을 반환한다.
