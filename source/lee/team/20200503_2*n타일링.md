# 2\*n 타일링

## 문제 접근 방법

- 처음엔 일일이 그려서 피보나치수열을 도출해냄
- 점화식 세우기

  <img src="imgs/lee/200503_2*nTiling.png">

### 코드

````java
class Solution {
  public int solution(int n) {
      if(n<=2) return n;
      int[] dp = new int[n];
      dp[0]=1;
      dp[1]=2;
      for(int i=2;i<n;i++){
          dp[i]=(dp[i-1]+dp[i-2])%1000000007;
      }
      return dp[n-1];
  }
}```

### Time complexity

O(n)

### Space complexity

O(n)
