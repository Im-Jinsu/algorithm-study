## 비밀지도
### 내가 고안한 방법


```java
public static String[] secretMap(int n, int[] arr1, int[] arr2) {
		String[] answer = new String[n];
		String ans = "";
		
		for(int i=0 ; i<n ; i++) {		
			ans = Integer.toBinaryString(arr1[i]|arr2[i]);
			int len = ans.length();
			if(ans.length()<5) {
				for(int j=0 ; j< 5-len ; j++)	ans = "0"+ ans;
			}
			answer[i] = ans.replace("1", "#").replace("0", " ");
		}
		return answer;
	}
```
