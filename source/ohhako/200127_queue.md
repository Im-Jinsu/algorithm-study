## 다리를 지나는 트럭
### 내가 고안한 방법
```java
import java.util.*;
class Solution {
    public int solution(int bridge_length, int weight, int[] truck_weights) {
        int sec=0;
        Queue<Integer> waiting_truck = new LinkedList();
        Queue<Integer> crossing_truck = new LinkedList();
        for(int t: truck_weights)   waiting_truck.add(t);
        
        while(!waiting_truck.isEmpty()){
        	int all_weight = 0;
        	sec++;
            //조건확인 (무게)
            for(Iterator<Integer> iter = crossing_truck.iterator();iter.hasNext(); )
            	all_weight += iter.next();
            //조건시 다리건너
            if((crossing_truck.size() <= bridge_length )
               && (all_weight+waiting_truck.peek() <=weight)){
            	int e = waiting_truck.poll();
                crossing_truck.add(e);
            }
            //트럭하나 건너는데 다리 길이만큼 걸림
            if(sec>bridge_length && (sec+1)%bridge_length==0)
            	crossing_truck.poll(); 
            //조건 확인(무게)
            all_weight = 0;
            for(Iterator<Integer> iter = crossing_truck.iterator();iter.hasNext(); )
            	all_weight += iter.next();     
        }
        return sec;
    }
}
```
