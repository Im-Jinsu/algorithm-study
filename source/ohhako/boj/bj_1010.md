# 백준 dp 문제

- 다리 놓기

## 풀이 방법

### 문제 이해하기

- 입력값: 테스트케이스 (t), t개의 n, m
- 출력값: 좌에 사이트 n개, 우측에 n개보다 적은 사이트 m개가 존재할때 다리가 겹치지 않게 최대로 사이트를 이어놓을 수 있는 경우의 수

### 문제 접근 방법

- DP문제를 접근할때 하나의 규칙을 두어 반복수행, 예외를 고려해야 한다.
- n개의 다리를 놓을 수 있는 경우는 최소

### 해결하지 못한 이유

- 조합공식을 알지 못했다.
